# YeoboSay 서비스 기능 개요

## 1. 서비스 소개
YeoboSay는 노인 사용자를 위한 AI 음성 대화 말벗 서비스입니다. 사용자는 스마트폰 앱에서 간편하게 음성 통화를 시작하고, AI가 자연스러운 대화를 이어가며 안부를 물어줍니다.

## 2. 주요 사용자 가치
- 노인 사용자의 외로움 완화
- 음성 중심 인터페이스로 쉬운 사용 경험 제공
- 실제 전화 통화와 유사한 자연스러운 대화 흐름
- 대화 세션과 대화 기록을 저장해 반복 사용 가능

## 3. 핵심 기능
### 3.1 AI 음성 통화 말벗
- 사용자가 앱에서 통화 버튼을 눌러 AI와 음성 대화를 시작
- AI가 음성을 인식하고 텍스트로 변환
- AI는 AIGC 모델을 사용해 적절한 응답을 생성
- 생성된 응답을 다시 음성으로 변환하여 사용자에게 전달

### 3.2 노인 친화적 UX
- 큰 글자와 명확한 화면 구성
- 단순한 통화 화면과 직관적인 조작 버튼
- 통화 중 대화 내용이 대화 기록 형태로 표시되어 이해 쉬움

### 3.3 통화 세션 관리
- 통화 시작 시 `call session`을 생성
- 각 발화 단위를 `call turn`으로 저장
- 통화 종료 후에도 대화 기록을 조회할 수 있음
- 동일한 세션에서 여러 번 대화 가능

### 3.4 자동 대화 흐름 지원
- ver0.2 계획에서는 자동 통화 시작과 AI 주도 대화 흐름을 강화
- 백엔드에서 대화 단계 오케스트레이션을 관리
- 사용자가 별도 녹음 버튼 없이 자연스럽게 대화할 수 있는 구조

### 3.5 향후 확대 가능한 기능
- 입출력 음성 스트리밍 기반 실시간 통화
- 수신 전화 형태의 자동 호출/수락 시나리오
- 대화 이력 기반 맞춤형 말벗 경험

## 4. 서비스 구성
### 4.1 Android 클라이언트
- 사용자 인터페이스 및 통화 화면 구현
- 음성 입력/출력 처리
- 백엔드 API 호출로 통화 세션 생성 및 대화 턴 전송
- 대화 기록 화면 표시

### 4.2 백엔드 서버
- NestJS 기반 REST API
- Prisma + PostgreSQL 데이터 저장소
- 대화 세션 및 턴 관리 API
- AI 대화 생성 및 음성 처리 연계 준비 구조

### 4.3 API 중심 설계
- `POST /call-sessions`: 통화 세션 생성
- `GET /call-sessions/{id}`: 세션 조회
- `GET /call-sessions/{id}/turns`: 대화 턴 목록 조회
- `POST /call-sessions/{id}/turns`: 사용자 발화 저장 및 AI 응답 생성

## 5. 기대 효과
- 기술 중심이 아닌 사용자 중심 대화 경험 제공
- 음성 기반 상호작용으로 접근성과 사용성을 높임
- 노인 사용자도 부담 없이 사용할 수 있는 말벗 서비스
- 실시간 전화형 대화로 사회적 고립 완화에 기여

## 6. 발표용 핵심 메시지
- YeoboSay는 노인 맞춤 AI 음성 말벗 플랫폼입니다.
- 간단한 통화 버튼만으로 AI가 대화를 이어갑니다.
- 모든 대화는 통화 세션으로 기록되어 나중에 다시 확인할 수 있습니다.
- 현재는 Android 앱 내부 AI 음성 통화 MVP 기반이며, 이후 자동 호출과 실시간 통화로 확장 가능합니다.
